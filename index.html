How It Works

Get your compliance score in minutes

1

Install

Add AIR Blackbox to your project with a single pip command.

pip install air-blackbox-mcp
2

Scan

Run the scanner against your Python AI agent codebase to detect compliance gaps.

from air_blackbox_mcp.scanner import scan_code
result = scan_code(open("your_agent.py").read())
print(result["compliance_score"])
3

Fix

Get remediation guidance and trust layer code for each failing article.

from air_blackbox_mcp import add_trust_layer
fixed_code = add_trust_layer(code, framework="langchain")
